MediaControl

What's that!?

Fire some js function on the css @media change

Set custom sizes & media

To set custom @media, just use this template:

CSS
@media only screen and (min-width: 8192px){
#MediaControl{z-index: 6}
#MediaControl:before{content: 'MonsterScreen';}
}
JS
$('#exp').on('onMonsterScreen', function() {
$(this).css({background: '#8800FF', color: '#000000'});
alert('I\'m full of pixels!');
})
